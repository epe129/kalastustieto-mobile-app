# kalastustieto-mobile-app

Tämä on mobiilisovellus, jossa on tkinter backend, josta voit lisätä kalasaaliin tiedot tietokantaan. jotka näytetään mobiilisovelluksessa.

# Tehty käyttäen:
- Flask
- React Native Expo
- SQL database
- Python tkinter

# Tarvitaan:
- XAMPP
- Python
- phpmyadmin database
- Node.js
- React Native Expo 
- Android studio/emulatori
